# STEP_3D Content Guide

## Где менять контент
- Главная структура страницы: `index.html`
- Карточки кейсов: `assets/js/cases.js`
- Базовые стили: `assets/css/styles.css`
- Компоненты/карточки/формы: `assets/css/components.css`
- Адаптив: `assets/css/responsive.css`

## Медиа-каталоги
- Кейсы: `assets/img/cases/`
- Партнеры: `assets/img/partners/`
- Иконки: `assets/img/icons/`
- OG-превью: `assets/img/og/`
- Временные заглушки: `assets/img/placeholders/`
- Видео: `assets/video/`
- Файлы для скачивания: `assets/files/`

## Рекомендации
1. Не добавляйте внешние зависимости/сборщики без необходимости.
2. Для GitHub Pages используйте только относительные пути.
3. Перед публикацией проверяйте якоря, фильтры, форму и mobile-меню.
