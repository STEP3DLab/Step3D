# Папка изображений сайта Step3D

Структура сделана так, чтобы было понятно, куда добавлять новые картинки.

## Что где хранить

- `Logo_Step3D.svg` — основной логотип сайта (используется в шапке, favicon, SEO/OG).
- `site/` — общие изображения сайта (баннеры, фоновые иллюстрации, иконки секций).
- `cases/` — изображения для кейсов (до/после, фото результатов, рендеры проектов).
- `team/` — фото команды, экспертов, спикеров.
- `clients/` — логотипы клиентов и партнеров.

## Рекомендации

- Названия файлов делайте латиницей и через дефис, например: `scan-medical-case-01.webp`.
- Для фотографий предпочтительно использовать `.webp`.
- SVG используйте для логотипов и векторных иконок.
