# STEP_3D — инженерная студия полного цикла

Статический сайт инженерной студии STEP_3D для публикации на GitHub Pages.

## Что внутри

- Современный одностраничный сайт в `docs/`.
- Акцент на конверсию в заявку: ясный оффер, услуги, кейсы, процесс, FAQ и форма.
- Отдельные юридические страницы: `offer.html` и `privacy.html`.

## Стек

- HTML5
- CSS3
- Vanilla JavaScript

Без сборщиков и фреймворков: проект разворачивается как набор статических файлов.

## Структура

```text
Step3D/
├── docs/
│   ├── index.html
│   ├── styles.css
│   ├── main.js
│   ├── robots.txt
│   ├── sitemap.xml
│   └── assets/
│       └── logo.svg
├── index.html
├── offer.html
├── privacy.html
├── README.md
└── package.json
```

## Локальный запуск

1. Откройте `docs/index.html` в браузере.
2. Или запустите локальный сервер:

```bash
npm run dev
```

После этого сайт будет доступен на `http://localhost:8080`.

## Публикация

GitHub Pages настроен на директорию `docs/`.

```bash
npm run deploy
```
