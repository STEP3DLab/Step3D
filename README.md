# STEP_3D

Финальная production-ready версия сайта для GitHub Pages.

## Структура

```text
Step3D/
├── index.html
├── assets/
│   ├── css/
│   │   └── main.css        # глобальные стили сайта
│   ├── js/
│   │   └── main.js         # основной интерактив и 3D-логика
│   └── media/              # изображения, иконки, модели, видео (добавляйте сюда)
├── README.md
└── .gitignore
```

### Куда что класть

- **Стили**: в `assets/css/main.css` (или рядом дополнительные CSS-файлы по секциям).
- **Скрипты**: в `assets/js/` (основной вход — `main.js`).
- **Медиа**: в `assets/media/` (изображения, видео, STL/GLB и пр.).

## Публикация

Сайт разворачивается из корня репозитория (ветка `main`, папка `/`).
Главная страница: `index.html`.
