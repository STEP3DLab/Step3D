# STEP_3D — инженерная студия полного цикла (статический сайт)

Репозиторий содержит статический одностраничный сайт STEP_3D для публикации на GitHub Pages.

## Стек

- HTML
- CSS
- Vanilla JavaScript

Без React/Vite/npm-сборки: сайт работает как обычные статические файлы.

## Структура

```text
Step3D/
├── docs/
│   ├── index.html      # основная страница
│   ├── styles.css      # стили
│   ├── main.js         # интерактивность
│   └── assets/
│       └── logo.svg
├── index.html          # редирект на docs/index.html
├── offer.html
├── privacy.html
└── README.md
```

## Локальный запуск

Достаточно открыть `docs/index.html` в браузере.

Для совместимости с привычным открытием корневого файла можно открыть `index.html` — он переадресует на актуальную версию.

## Контент, требующий финальной подстановки

В `docs/index.html` оставлены человекочитаемые временные контакты (Telegram, email, телефон, адрес) и могут быть заменены на подтверждённые данные.
